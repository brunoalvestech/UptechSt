# Uptech_SolucoesTecnologicas
UPTECH – Gestão de Tecnologia para Escolas Privadas
A UPTECH é uma landing page de alta performance desenvolvida para apresentar soluções de gestão tecnológica e infraestrutura de TI especificamente para o setor educacional. O projeto foca em um design futurista com uma experiência de usuário (UX) fluida.

Demonstração
Você pode visualizar o projeto online aqui: [[Link para o seu GitHub Pages]](https://brunoalvestech.github.io/UptechSt/)

Funcionalidades
Design Futurista: Interface baseada em "Dark Mode" com elementos visuais modernos, como fundos em grid e efeitos de brilho (glow).

Interface Responsiva: Totalmente adaptado para dispositivos móveis através de Media Queries.

Animações Dinâmicas: Utilização de Intersection Observer para efeitos de "Fade-Up" conforme a rolagem da página.

Componentes Interativos:

Navegação com efeito de desfoque (backdrop-filter).

Cards de soluções com efeitos de hover e bordas animadas.

Simulador de terminal de comando na seção "Bastidores".

Integração de Contato: Formulário funcional que valida dados e redireciona o usuário diretamente para o WhatsApp com uma mensagem pré-formatada.

Tecnologias Utilizadas
HTML5: Estruturação semântica de todas as seções.

CSS3 (Custom Properties): Uso extensivo de variáveis CSS para paleta de cores e temas, além de animações complexas em @keyframes.

JavaScript (Vanilla): Lógica para o menu mobile, animações de entrada e integração com a API do WhatsApp.

Google Fonts: Fontes Anek Bangla, Open Sans e Roboto para tipografia refinada.

Estrutura do Projeto
O site é composto pelas seguintes seções principais:

Hero: Destaque principal com estatísticas de impacto.

Problemas/Riscos: Identificação de dores comuns em TI escolar.

Soluções: Cards detalhando os serviços oferecidos.

Diagnóstico: Passo a passo da consultoria técnica.

Planos: Tabela de serviços segmentada.

Contato: Formulário integrado para captação de leads.
